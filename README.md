# 极客时间评论区
演示地址：
+ https://geektime-comment.red-notes.com/
+ https://geektime.pages.dev/

## 技术栈
vue + daisyui(tailwindcss) + axios

## 功能
+ 无限制查看评论区
+ 支持夜晚模式
+ 只显示作者回复的评论
+ 使用 localStorage 存储专栏
